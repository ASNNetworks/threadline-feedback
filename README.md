# Threadline Feedback & Issues

This is the public issue tracker for **Threadline** — the AI Chat Supercharger browser extension.

Threadline enhances your AI chat experience on **Claude**, **ChatGPT**, and **Gemini** with powerful features like branch navigation, reasoning visualization, and conversation export.

## How to use this repo

This repository doesn't contain any source code. It exists so you can:

- **Report bugs** you've encountered
- **Request features** you'd like to see
- **Ask questions** about Threadline
- **Share feedback** and ideas

## Before opening an issue

1. **Search existing issues** to avoid duplicates
2. **Use the appropriate label** (bug, feature request, question)
3. **Be as specific as possible** — include browser, OS, and steps to reproduce for bugs
